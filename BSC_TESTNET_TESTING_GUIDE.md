1. Visit https://testnet.bnbchain.org/faucet-smart

4. Wait ~30 seconds for transaction confirmation
5. Check your wallet balance

#### Option B: Use PancakeSwap Testnet

## üß™ Testing Checklist

### 1. Swap Interface Testing

#### Test Case 1: BNB ‚Üí Token Swap
**Steps**:
1. Navigate to http://localhost:10002/dex
2. Connect wallet (MetaMask recommended)
3. Ensure network is "BSC Testnet"
4. Select "Swap" tab
5. From: BNB
6. To: USDT
7. Enter amount: 0.01 BNB
8. Review quote (should show estimated USDT output)
9. Click "Swap"
10. Confirm transaction in wallet
11. Wait for confirmation (~3 seconds)
12. Verify balance updated

**Expected Results**:
- ‚úÖ Quote appears within 2 seconds
- ‚úÖ Transaction succeeds
- ‚úÖ Balance updates correctly
- ‚úÖ Transaction visible on BSCScan

#### Test Case 2: Token ‚Üí BNB Swap
**Steps**:
1. From: USDT
2. To: BNB
3. Enter amount: 1 USDT
4. Check if approval needed
5. If needed: Click "Approve USDT" ‚Üí Confirm
6. Wait for approval confirmation
7. Click "Swap"
8. Confirm transaction
9. Verify balance updated

**Expected Results**:
- ‚úÖ Approval transaction succeeds (if needed)
- ‚úÖ Swap transaction succeeds
- ‚úÖ BNB balance increases
- ‚úÖ USDT balance decreases

#### Test Case 3: Token ‚Üí Token Swap
**Steps**:
1. From: USDT
2. To: BUSD
3. Enter amount: 1 USDT
4. Verify quote (should be ~1 BUSD due to peg)
5. Approve if needed
6. Execute swap
7. Verify balances

**Expected Results**:
- ‚úÖ Route through WBNB (USDT ‚Üí WBNB ‚Üí BUSD)
- ‚úÖ Quote approximately 1:1
- ‚úÖ Slippage protection works
- ‚úÖ Transaction succeeds

#### Test Case 4: Slippage Testing
**Steps**:
1. Set slippage to 0.1%
2. Try swapping 0.01 BNB ‚Üí USDT
3. If transaction fails, increase slippage to 0.5%
4. Retry

**Expected Results**:
- ‚úÖ Low slippage may fail (expected)
- ‚úÖ Higher slippage succeeds
- ‚úÖ Actual slippage within tolerance

## üêõ Common Issues & Solutions

### Issue 1: "Insufficient BNB for gas"
**Solution**: Get more tBNB from faucet

### Issue 2: "Transaction failed"
**Possible Causes**:
- Slippage too low ‚Üí Increase to 1-2%
- Network congestion ‚Üí Wait and retry
- Insufficient allowance ‚Üí Re-approve token

### Issue 3: "Pair does not exist"
**Solution**:
- Create the pair by adding initial liquidity
- Or use a different token pair that already exists

### Issue 4: Quotes not loading
**Possible Causes**:
- RPC endpoint slow ‚Üí Wait 10 seconds
- Pool doesn't exist ‚Üí Try different token pair
- Browser cache ‚Üí Hard refresh (Cmd+Shift+R)

### Issue 5: Wallet won't connect
**Solution**:
1. Make sure MetaMask is installed
2. Add BSC Testnet to MetaMask manually:
   - Network Name: BSC Testnet
   - RPC URL: https://data-seed-prebsc-1-s1.binance.org:8545/
   - Chain ID: 97
   - Currency Symbol: tBNB
   - Block Explorer: https://testnet.bscscan.com
